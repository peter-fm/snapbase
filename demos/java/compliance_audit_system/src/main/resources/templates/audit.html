How to Generate Audit Logs
To see actual audit logs:
  1. Run the compliance demo from the Dashboard
  2. After the demo completes, check the compliance_workspace/audit_exports/ directory
  3. Use the export buttons above to download the generated audit files
Sample Audit Log Format:
timestamp,user,action,details
